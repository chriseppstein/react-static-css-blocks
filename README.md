# React-Static - Basic Example

This example is the most basic version of react-static available. It includes:
- Babel
- SASS/SCSS imports
- Image imports
- File imports
- SASS Loader
- Automatic Routing
- Experimental CSS blocks configuration (WIP)

To get started, clone this repo, then run `yarn start`
